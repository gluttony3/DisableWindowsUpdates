⚙️ Windows Updates Disabler - Повний набір інструментів
========================================================

📦 ВМІСТ ПАКЕТУ
================

Цей пакет містить **5 способів** для відключення/включення Windows оновлень:

МЕТОД 1️⃣ : BATCH ФАЙЛИ (РЕКОМЕНДУЄТЬСЯ)
─────────────────────────────────────────
✅ Найпростіший в використанні
✅ Не потребує встановлення Python або PowerShell
✅ Прямо запускаються на Windows

📄 disable_windows_updates.bat
   └─ Відключити оновлення (версія Batch)
   └─ Запуск: Праву кнопку → "Запустити від адміністратора"

📄 enable_windows_updates.bat
   └─ Включити оновлення назад (версія Batch)
   └─ Запуск: Праву кнопку → "Запустити від адміністратора"

МЕТОД 2️⃣ : POWERSHELL(РЕКОМЕНДУЄТЬСЯ ДЛЯ РОЗШИРЕНИХ КОРИСТУВАЧІВ)
──────────────────────────────────────────────────────────────────
✅ Найбільш потужний метод
✅ Інтерактивний формат
✅ Детальні повідомлення про помилки

🔵 Disable-WindowsUpdates.ps1
   └─ Відключити оновлення (версія PowerShell)
   └─ Запуск: powershell -ExecutionPolicy Bypass -File "Disable-WindowsUpdates.ps1"

🔵 Enable-WindowsUpdates.ps1
   └─ Включити оновлення назад (версія PowerShell)
   └─ Запуск: powershell -ExecutionPolicy Bypass -File "Enable-WindowsUpdates.ps1"

МЕТОД 3️⃣ : PYTHON (ДЛЯ РОЗРОБНИКІВ)
─────────────────────────────────────
✅ Кроз-платформний код
✅ Легко переносити на інші системи
✅ Легко редагувати під своїх потребах

🐍 disable_windows_updates.py
   └─ Відключити оновлення (версія Python)
   └─ Запуск: python disable_windows_updates.py

🐍 enable_windows_updates.py
   └─ Включити оновлення назад (версія Python)
   └─ Запуск: python enable_windows_updates.py
   └─ Вимога: Встановлений Python 3.7+

МЕТОД 4️⃣ : МЕНЮ (НАЙЗРУЧНІШЕ)
───────────────────────────────
✅ Інтерактивне меню для вибору
✅ Усі інструменти в одному місці
✅ Показує статус послуг

📋 Menu.bat
   └─ Головне меню для управління оновленнями
   └─ Запуск: Двічі клацніть на файлі

🚀 ШВИДКИЙ СТАРТ
════════════════

🎯 НАЙПРОСТІШИЙ СПОСІБ (1 клік):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Відкрийте Menu.bat (двічі клацніть)
2. Введіть "1" для відключення або "2" для включення
3. Дочекайтесь завершення
4. Перезавантажте комп'ютер

🎯 СПОСІБ 2 (Batch):
━━━━━━━━━━━━━━━━━━━
1. Клацніть правою кнопкою на disable_windows_updates.bat
2. Виберіть "Запустити від адміністратора"
3. Дайте дозвіл коли буде запит
4. Перезавантажте комп'ютер

🎯 СПОСІБ 3 (PowerShell):
━━━━━━━━━━━━━━━━━━━━━━━
1. Відкрийте PowerShell від адміністратора
2. Перейдіть в папку зі скриптами
3. Виконайте: powershell -ExecutionPolicy Bypass -File "Disable-WindowsUpdates.ps1"
4. Перезавантажте комп'ютер

🎯 СПОСІБ 4 (Python):
━━━━━━━━━━━━━━━━━━━━
1. Переконайтеся, що Python встановлений
2. Відкрийте Command Prompt від адміністратора
3. Перейдіть в папку зі скриптами
4. Виконайте: python disable_windows_updates.py
5. Перезавантажте комп'ютер


✅ ПЕРЕВІРКА УСПІШНОГО ВИКОНАННЯ
══════════════════════════════════

Після запуску скрипту перевірте:

1. Служба Windows Update зупинена:
   └─ Відкрийте Services (services.msc)
   └─ Знайдіть "Windows Update"
   └─ Статус повинен бути "Stopped" або порожньо

2. Немає автоматичних оновлень:
   └─ Перейдіть в Settings → Update & Security
   └─ Не повинно бути активних оновлень

3. Точка відновлення створена:
   └─ Control Panel → System Protection
   └─ Повинна бути точка "Before disabling Windows Updates"

❓ ПРОБЛЕМИ?
═════════════

Помилка: "Access Denied"
└─ Запустіть від адміністратора (правою кнопкою → Run as admin)

Помилка: "File not found"
└─ Переконайтеся, що знаходитесь в правильній папці

Не працює після запуску:
└─ ПЕРЕЗАВАНТАЖТЕ комп'ютер!
└─ Зміни потребують перезавантаження Windows

Хочу включити назад:
└─ Запустіть enable_windows_updates.bat або Enable-WindowsUpdates.ps1


🔐 БЕЗПЕКА
═══════════

Ризики:
 ⚠️  Без оновлень система менш захищена
 ⚠️  Деякі программи можуть не працювати
 ⚠️  Можуть виникнути проблеми сумісності

Як зберегти безпеку:
 ✅ Користуйтеся хорошим антивірусом
 ✅ Вручну включайте оновлення щомісяця
 ✅ Отримуйте принаймні критичні патчи
 ✅ Запускайте Windows Firewall

Рекомендована стратегія:
 1. Запустіть disable скрипт
 2. Користуйтеся ПК як завгодно
 3. Щомісяця запустіть enable скрипт
 4. Встановіть критичні оновлення
 5. Запустіть disable скрипт знову

🎓 ЯК ЦЕ ПРАЦЮЄ (СПРОЩЕНО)
════════════════════════════

Windows оновлення працюють через:

┌─────────────────┐
│  Task Scheduler │ ← Запускає оновлення за розписом
└────────┬────────┘
         │
┌────────▼────────┐
│  Update Service │ ← wuauserv - головна служба
└────────┬────────┘
         │
┌────────▼────────┐
│  Download Svcs  │ ← BITS, DoSvc - завантаження
└────────┬────────┘
         │
┌────────▼────────┐
│  Registry Deps  │ ← Параметри в реєстрі
└─────────────────┘

Наш скрипт вимикає ВСІ ці рівні:
✅ Зупиняє служби
✅ Деактивує завдання
✅ Змінює реєстр
✅ Застосовує політики
